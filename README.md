# File Organizer Script

##  Internship Project - Syntexchub

##  Description
This project is a Python-based file organizer that automatically sorts files into folders based on their file type.

##  Features
- Organizes files by extension (e.g., .jpg, .pdf, .txt)
- Creates folders automatically
- Improves file management efficiency

##  Technologies Used
- Python

## How to Run
1. Run the Python script
2. Provide the folder path
3. Files will be automatically organized
